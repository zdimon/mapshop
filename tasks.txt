Задачи.

1. Создать раздел списка товаров. Организовать сортировку по популярности и цене. Обеспечить фильтр по категориям.


2. Страница карточки товара с главным фото, описанием и ссылкой на просмотр остальных фоток (галереи).


3. Страница выбора киоска. Прикрутить карту google api. Поотмечать киоски.
 Пример тут http://telek.kardos.media/distribution/
 При клике на киоск делать http запрос, куда передать параметры: id киоска и id корзины (скрытое поле).
 Подсветить выбранный киоск. Загрузить командой киоски из csv файла.

4. Раздел уведомления. Создать страницу с формой с емейлом и телефоном. При сабмите сохранять инфу в базе. Вызвать задачу рассылки при изменении статуса товара с нет в наличии в есть в наличии.


5. Создать API запросы, меняющие статус товара с есть в наличии на нет и наоборот с запуском задач на емейл и СМС информирование.

6. Корзина. Добавление в корзину. Ссылка с редиректом на ту же страницу и добавлением товара в корзину. Вывод содержимого корзины. Возможность удаления из корзины. Ссылка "Оплата заказа" c переходом на страницу выбора киоска и передачей id корзины. Учитывать сумму заказа при активации ссылки.

7. Форма личных данных (два варианта в зависимости от параметра физ и юр лица). Сохранение данных в базу. API запрос на создание пользователя на сайте- родителе. Переход на страницу успешного создания заказа. Отправка сообщения на мыло пользователя с инфой по заказу. Отправка уведомления администрации.
Запуск задач на СМС информирование.


8. СМС информирование.
